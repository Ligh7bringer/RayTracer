A simple ray tracer based on [Ray Tracing in a Weekend](http://in1weekend.blogspot.com/).

# Building

1. Clone this repo and ``cd RayTracer``

2. `` mkdir build && cd build ``

3. ``cmake ../``

4. Use ``make``, your IDE , or something else to build the executable
